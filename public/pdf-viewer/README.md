# IMPA PDF 查看器

## 📚 功能特性

- ✅ 在线PDF预览
- ✅ 文件列表浏览
- ✅ 一键下载
- ✅ 新窗口打开
- ✅ 响应式设计
- ✅ 键盘快捷键支持

## 🚀 使用方法

1. 打开 `index.html` 文件
2. 从左侧列表选择要查看的PDF文件
3. 使用右侧的按钮进行下载或新窗口打开

## ⌨️ 快捷键

- `ESC`: 关闭当前PDF
